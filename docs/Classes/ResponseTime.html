ResponseTime

public class ResponseTime: RouterMiddleware

A Kitura ‘RouterMiddleware’ that reports the time (in milliseconds) that a request took in one of the HTTP response headers (X-Response-Time by default).

Note

In order to get correct numbers it is important to place this middleware as early as possible in the list of handlers and middleware of your application. Any time spent in handlers or middleware that run before this middleware is invoked will not be counted.
